Welcome to Westminster Bookstore API

This is a RESTful API for managing a bookstore. Use the following endpoints to interact with the API:

  • GET /api/books - Retrieve all books
  • POST /api/books - Create a new book
  • GET /api/authors - Retrieve all authors
  • POST /api/authors - Create a new author
  • GET /api/customers - Retrieve all customers
  • POST /api/customers - Create a new customer
  • GET /api/carts/{customerId} - Retrieve a customer's cart
  • POST /api/carts - Add items to a cart
  • GET /api/orders - Retrieve all orders
  • POST /api/orders - Place a new order

Use a tool like Postman to test these endpoints.
